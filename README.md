# Facial Mood Detection using OpenCV

This project implements a facial emotion recognition system using OpenCV and the CK+ dataset.

## Features
- Face detection using Haar Cascade
- Emotion classification using LBPH
- Real-time webcam-based mood detection
- Interactive OpenCV UI

## How to Run
1. Train the model using the notebook
2. Run `realtime_ui.py` for real-time detection

## Tech Stack
- Python
- OpenCV
- NumPy
